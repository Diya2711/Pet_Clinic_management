# 🐶 Flutter Pet Clinic App

A modern Flutter-based mobile application designed to simplify pet healthcare management. Users can explore veterinary services, view pet information, and book appointments with ease. The app also includes an admin panel with real-time data powered by Firebase.

## ✨ Features

* Browse veterinary services (Vaccination, Grooming, Surgery, etc.)
* View detailed pet care information
* Book appointments through an interactive form
* Admin dashboard to manage and track bookings
* Real-time data updates using Firebase

---

## 🛠️ Tech Stack

* **Flutter** – Frontend UI framework
* **Dart** – Programming language
* **Firebase** – Backend & Real-time Database
